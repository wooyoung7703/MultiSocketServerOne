롯데카드 클러치 유의사항

  • 이용대상

    본인명의의 휴대폰을 이용하고 계시는 고객님이라면, 휴대폰 인증 절차를 통해 누구나 회원가입이 가능합니다.

  • 이용가능 스마트폰

    안드로이드, IOS가 탑재된 스마트폰에서 이용하실 수 있습니다.

    *이용가능 OS버전 : 안드로이드 2.3 이상 / IOS 5.0 이상(단, 3G/4G 요금제 가입 없이 Wifi 환경으로 이용하는 스마트폰 공기계나 태블릿PC에서는 이용이 불가능합니다.)

  • 도난/분실 시 부정사용 차단

    부정사용 방지를 위해 최초 가입시 서비스 이용 비밀번호 4자리를 설정하게 되며, 결제 서비스(앱카드 현장결제) 이용시 추가로 결제 비밀번호 6자리를 이용합니다.

    스마트폰 도난/분실의 경우, 비밀번호를 입력해야 사용이 가능하기 때문에 타인이 부정사용할 수 없습니다.
    (비밀번호를 연속 5회 오류 입력 시 앱이 잠금상태로 변환되며, 본인인증과 비밀번호 재설정을 통해 다시 이용 가능합니다.)

  • 분실신고 시 결제 차단

    클러치에 등록하신 카드를 분실하여 롯데카드로 분실신고를 하신 경우, 해당 카드로는 앱카드(간편결제) 결제가 자동 차단됩니다.

  • 결제 서비스 이용

    클러치 내의 바로결제, 카드결제 서비스를 이용하기 위해서는 '롯데앱카드(간편결제)' 설치 및 가입이 반드시 필요합니다. 롯데앱카드(간편결제)는 마켓(안드로이드 : 플레이스 스토어 / 아이폰 : 앱 스토어)에서 다운로드 가능하며, 본인명의의 핸드폰과 롯데카드를 이용하시는 고객님이라면 누구나 가입 가능합니다.

  • 기타

    기기변경, 번호변경, 번호이동 등 휴대폰 정보가 변경되는 경우, 앱 삭제 후 재설치를 통해 정보변경을 권장합니다. 앱 삭제 후 재설치 시 간단한 인증절차를 통해 재가입이 가능하며, 기존 이용내역 및 결제내역은 보존됩니다.

    데이터 네트워크가 원활하지 않거나 카드사 시스템 장애로 결제가 어려운 경우 플라스틱 카드로 결제하시기 바랍니다.

    추가 문의사항은 고객센터로 연락 바랍니다.
